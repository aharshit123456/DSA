# Sparse Matrix

## Write a program to calculate the multiplication of 2 Matrics

## Find an efficient matrix representation

```c

// S[m][n] -> Sparse Matrix where m = rows, n = columns

int count = 0;
for(int i = 0; i < m; i++){
    for(int j = 0; j < n; j++){
        if(S[i][j] != 0){
            count++;
        }
    }
}

int E[count][3];
int k = 0;
for(int i = 0; i < m; i++){
    for(int j = 0; j < n; j++){
        if(S[i][j] != 0){
            E[k][0] = S[i][j];
            E[k][1] = i;
            E[k][2] = j;
            k++;
        }
    }
}
```
